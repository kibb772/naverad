# 알려조 (allyojo.com) 기능 분석 리포트

**분석 일자**: 2026-04-22
**분석 대상**: https://www.allyojo.com
**서비스 요약**: AI 기반 광고 성과 자동 분석 서비스 (Meta, Naver, Google Ads 통합)
**운영사**: 주식회사 프라임허브 (대표 김경훈)

---

## 1. 서비스 개요

알려조는 Meta(Facebook/Instagram), 네이버 검색광고, Google Ads 3대 광고 매체의 성과 데이터를 자동 수집·분석하여 이상 탐지, 원인 진단, 개선 제안을 제공하는 AI 기반 SaaS 서비스입니다.

---

## 2. 핵심 기능 목록

### 2.1 인증 및 사용자 관리
- 카카오 OAuth 로그인
- 이메일/비밀번호 로그인
- 회원가입 (이메일, 비밀번호, 이름, 전화번호)
- 약관 동의 (서비스 이용약관, 개인정보 처리방침, 광고 데이터 수집 동의, 마케팅 수신 동의)

### 2.2 광고 매체 연동
- Meta (Facebook/Instagram) 광고 계정 연동
- 네이버 검색광고 계정 연동
- Google Ads 계정 연동
- OAuth 또는 API 키 기반 연동
- 연동 소요 시간: 약 3분

### 2.3 통합 대시보드
- 3대 매체 통합 성과 대시보드
- 핵심 지표 실시간 표시: 노출수, 클릭수, CTR, CPC, 전환율, ROAS
- 플랫폼별 오늘 성과 요약 (지출, ROAS, 변동률)
- 캠페인/광고그룹/소재 단위 상세 분석
- 매체 간 성과 비교
- 실시간 데이터 동기화

### 2.4 AI 이상 탐지
- CTR/CPC/전환율 등 핵심 지표 급변 감지
- 과거 데이터 기반 기준값 자동 학습 (계정 히스토리 + 업종 벤치마크)
- 위험도별 심각도 분류 (주의/경고/위험)
- 이상 감지 시 대시보드 + 이메일 즉시 알림
- 예시: "Meta 캠페인 CPC가 기준값 대비 +32% 상승"

### 2.5 AI 원인 진단
- 이상 감지 시 자동 원인 분석
- 원인 분류: 오디언스 피로도, 입찰 경쟁 증가, 소재 CTR 하락, 시즌 영향 등
- 각 원인별 확률(%) 표시
- 과거 유사 사례 매칭
- 매체별 특성 반영 맞춤 진단
- 자연어 기반 전문가 수준 설명

### 2.6 AI 개선 제안 (권장 액션)
- 구체적 액션 아이템 제시 (소재 교체, 입찰가 조정, 예산 재배분 등)
- 우선순위별 정리
- 예상 개선 효과 추정
- 실행 가이드 제공

### 2.7 AI 성과 리포트
- GPT-4.1 기반 AI가 캠페인 데이터를 한글 문장으로 요약
- 일간/주간/월간 리포트
- 성과 등급 부여 (매우 좋음/좋음/보통/경고/위험)
- 변화 요인 분석
- 내일 액션 플랜 포함
- 대행사용 클라이언트 보고서 자동 생성
- 커스텀 브랜딩 지원 (로고, 색상)

### 2.8 통합 캠페인 관리
- 3대 매체 캠페인 한 화면에서 관리
- 캠페인 ON/OFF 토글
- 예산 조정
- 새 캠페인 생성
- 광고 만들기 기능

### 2.9 경쟁사 분석
- Meta Ad Library 기반 경쟁사 광고 수집
- 경쟁사 활성 광고 건수 추적
- AI 분석: 타겟, 톤, 오퍼, CTA 패턴 분석
- 인사이트 제공 (할인 오퍼 중심, 타겟 연령/성별, CTA 유형 등)

### 2.10 알림 시스템
- 대시보드 내 알림
- 이메일 알림
- 카카오톡 알림톡 (매일 아침 자동 발송)
- 알림 기준 커스터마이징

### 2.11 AI 카피 생성
- 광고 소재 카피 자동 생성 (Pro 플랜 이상)

### 2.12 예산 최적화 시뮬레이션
- 예산 배분 시뮬레이션 (Pro 플랜 이상)

## 4. 기술 스택 추정

- **프론트엔드**: Next.js (URL 패턴 `/_next/image` 확인)
- **인증**: 카카오 OAuth, 이메일/비밀번호
- **AI 엔진**: GPT-4.1 기반 (서비스 소개 페이지에 명시)
- **광고 API 연동**: Meta Marketing API, Naver Search Ads API, Google Ads API
- **알림**: 카카오 알림톡 API, 이메일

## 5. 사용자 플로우

```
회원가입/로그인
    ↓
광고 매체 연동 (OAuth/API 키)
    ↓
데이터 자동 수집 (캠페인/광고그룹/소재)
    ↓
AI 분석 & 이상 탐지 (24시간 모니터링)
    ↓
원인 진단 & 개선 제안
    ↓
알림 & 리포트 발송 (대시보드/이메일/카카오톡)
```

## 6. 타겟 사용자

1. **소상공인/자영업자**: 광고 성과 확인 시간·전문 지식 부족
2. **마케팅 대행사**: 다수 클라이언트 관리, 리포트 작성 시간 절감
3. **인하우스 마케터**: 여러 매체 통합 관리, 데이터 기반 의사결정
